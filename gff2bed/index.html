gff2bed

发表于 2017-08-24 | 分类于 project | 阅读次数

BED 文件格式提供了一种灵活的方式来定义的数据行，以用来描述注释的信息。BED行有3个必须的列和9个额外可选的列。 每行的数据格式要求一致。通常所见的bed文件为6列，但是一些软件要求输入12列bed文件(UCSC Genome Browser)，通过探索研究出一下两种从gff3/gtf转为12列bed文件的方法。
