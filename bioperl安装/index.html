bioperl安装(无需root权限)

发表于 2018-06-12 | 分类于 software | 阅读次数
  • ###bioperl 安装（cpan）
    1) perl -MCPAN -e shell 一路回车，安装完成后q退出

2) 下载bioperl(wget https://github.com/bioperl/bioperl-live/archive/master.zip)并解压：unzip master.zip
3) 进入解压后的目录安装

中间一直敲回车即可，若无报错，则安装完成

  • ##blastxml安装
    1) 下载blastxml(wget http://search.cpan.org/CPAN/authors/id/C/CJ/CJFIELDS/Bio-SearchIO-blastxml-1.70.tar.gz)并解压
    2) 进入解压后的目录安装
